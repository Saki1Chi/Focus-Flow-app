API Reference

Endpoints available for the Flutter mobile app

Tasks
GET /api/tasks List all tasks
Query params: date, status, mode, category_id
POST /api/tasks Create a task
POST /api/tasks/bulk Bulk upsert (sync from app)
Accepts an array of tasks. Creates new ones and updates existing ones (by ID).
PUT /api/tasks/{id} Update a task
DELETE /api/tasks/{id} Delete a task
Categories
GET /api/categories List categories
POST /api/categories Create category
PUT /api/categories/{id} Update category
DELETE /api/categories/{id} Delete category
Stats
GET /api/stats Task counts by status
GET /docs Interactive Swagger UI

Flutter Integration

Add this to pubspec.yaml:

dependencies:
  http: ^1.2.2

Android emulator → server address:

http://10.0.2.2:8000

Physical device (same WiFi):

http://<your-local-ip>:8000

Run the server:

cd backend
pip install -r requirements.txt
python main.py
